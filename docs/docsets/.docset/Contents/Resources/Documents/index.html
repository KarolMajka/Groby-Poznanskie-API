Poznan Graves

iOS app that use Poznan API to show list of dead people in Poznań

Installation

If you don’t have CocoaPods:

$ [sudo] gem install cocoapods

Then in project file type:

$ pod install

Pods used

  • Alamofire
  • Alamofire Object Mapper
  • Realm
